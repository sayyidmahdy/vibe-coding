buat project baru di folder ini

menggunakan next js

dan gunakan dependencies:

- tailwind css
- typescript
- prisma
- supabase

jangan terlalu low level atau detail
cukup instruksikan secara high level

nanti dokumen planningnya akan digunakan oleh programmer atau model yang lebih murah untuk diimplementasikan

----------------------
buatkan issue.md yang berisi perencanaan untuk nanti diimplementasikan oleh junior programmer atau ai model yang lebih murah

buat table users:
id integer auto increment
name varchar(255)
email varchar(255)
password varchar(255) (password merupakan hash dari bcrypt)
created_at timestamp default current_timestamp

buatkan api untuk registrasi user baru
endpoint : POST api/users

request body:
{
    "name": "budi",
    "email": "[EMAIL_ADDRESS]",
    "password": "[PASSWORD]"
}

response success:
{
    "id": 1,
    "name": "budi",
    "email": "[EMAIL_ADDRESS]",
    "created_at": "2022-01-01T00:00:00.000Z"
}

response error jika email sudah terdaftar:
{
    "message": "Email already exists"
}

response error jika email tidak valid:
{
    "message": "Invalid email"
}

response error jika password terlalu pendek:
{
    "message": "Password must be at least 6 characters"
}

response error jika ada error lain:
{
    "message": "Internal server error"
}

struktur folder di dalam src: 
- routes : ini berisi routing api next js
- services : ini berisi logic bisnis
- lib : ini berisi library
- utils : ini berisi utilitas
- types : ini berisi types  

struktur file
- routes : menggunakan format misal users-routes.ts
- services : menggunakan format misal users-service.ts
- lib : menggunakan format misal prisma-lib.ts
- utils : menggunakan format misal validator.ts
- types : menggunakan format misal user.ts

jelaskan tahapan-tahapan dalam mengimplementasikan api ini secara singkat, anggap nanti yang mengimplementasikan adalah junior programmer atau ai model yang lebih murah
